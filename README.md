## Weather Forecast App

### Working:
* Set up the latest version of [nodeJS](https://nodejs.org) (6 major version)
* `npm i`

###Building
* `npm run build`

###Running
* `npm run start`

### Lint
* `npm run test:lint` - lint codebase

### Style Guide
* JavaScript: https://github.com/airbnb/javascript
* CSS: https://github.com/airbnb/css
